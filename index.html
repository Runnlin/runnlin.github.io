关于我

我是 Runnlin，来自广东，就职于头部汽车电子企业 德赛西威（DesaySv），专注于 Android 系统底层与车载多媒体 方向的工程研发。

在 Android 领域，我深入研究 Android Framework 内核机制，包括 Binder IPC 架构、AIDL 跨进程通信、JNI 层接口设计，以及 MediaCodec 硬编解码与音视频同步等核心技术。同时具备 FFmpeg + OpenGL ES 软解方案的完整实战经验。

系统编程领域，我掌握 Linux 内核内存管理（虚拟地址空间、伙伴系统、Slab 分配器）以及 QNX Neutrino RTOS 的消息传递 IPC 机制，擅长面向车规级软件的实时性与可靠性设计。

此外，我具备 Spring + Vue 全栈 开发能力，正在系统学习 Rust 以提升系统级编程的安全性与性能边界。相信 深入一层，方能稳固上层 — 越接近底层，越能写出真正可靠的软件。

近期项目

vlc-android Kotlin · C++

基于 VLC 开源播放器进行深度定制，研究 libvlc 底层解码流程与 Android MediaPlayer 的集成架构，实现车载场景下的高兼容性多媒体播放。

LearningVideo Kotlin · FFmpeg · OpenGL ES

Android 音视频全栈学习工程，覆盖硬解码（MediaCodec）、FFmpeg 软解、OpenGL ES 渲染、OpenSL ES 音频播放与音视频同步等核心环节。

RunLocalmedia Kotlin · Android

本地媒体播放器 App，支持视频列表管理与自定义播放器，深入实践 Android Jetpack 组件与 Room 数据库设计。

myAIDL Kotlin · AIDL · Binder

Android Binder IPC 完整 Demo，演示 AIDL 接口定义、跨进程服务绑定、IBinder 对象传递与死亡通知（linkToDeath）等核心机制。

TriShop Java · Spring · Vue

全栈电商平台，后端基于 Spring Boot 构建 RESTful API，前端使用 Vue 实现响应式 SPA，覆盖商品管理、购物车与订单全流程。
